Angular Cd-Tabs !

Introduction

cd-tabs is a simple Angular CLI component that manage tabs, based on Ionic tabs.

cd-tabs is able to:

  • display horizontal and vertical tabs.
  • manage icons.
  • manage router integration or simple tabs.

Installation

You can install cd-tabs using NPM:

npm install angular-cd-tabs

Then, you shall import the CdTabsModule in app.module.ts.
