Components API

The CdTabsComponent component is organized under several sub-components and interfaces:

  • CdTabBarComponent: which list all buttons used by tabs definition.
  • CdTabButtonComponent: a tab button definition.
  • CdTabContentComponent: a tab content definition.
  • CdTabInterface: a tab interface for interaction with component.
CdTabsComponent Selector: cd-tabs The CdTabsComponent component is the entry point of the tabs definition.

Properties

selectMode
Description String: set-up the autoselection mode at initialization:
  • config: use the configuration to auto select the activated tab (default).
  • router: use router path matching to auto select the activated tab.
displayMode
Description String: set-up the display mode of tabs:
  • default: display tabs in default format.
  • segments: display tabs as Ionic Segments.
disposition
Description String: set-up the disposition of tabs:
  • left: align tab buttons on left side (default).
  • center: align tab buttons on center side.
  • right: align tab buttons on right side.
  • vertical: display tab buttons in vertical bar.

Methods

select
Description Select a tab by the reference index number, or tab id or CdTabButtonComponent reference.
Signature select(tab: number | string | CdTabButtonComponent) => Promise<boolean>
getSelected
Description Get the currently selected tab.
Signature getSelected() => Promise<CdTabInterface | null>
getTab
Description Get a specific tab by the reference index number of its tab property, the tab id or the CdTabButtonComponent.
Signature getTab(tab: number | string | CdTabButtonComponent) => Promise<CdTabInterface | null>

Events

tabChangedEvent

Emitted when the tab has changed.

Emit the CdTabInterface event.
CdTabBarComponent Selector: cd-tab-bar The CdTabBarComponent component allow you to define and structure all tab buttons for your implementation.

Properties

color
Description

String: define the Ionic color used for tab-bar.

You can define CSS variables for standalone integration.

CSS variables

--cd-color Color of the tab button font.
--cd-color-selected Color of the tab button font when the tab is selected.
--cd-color-hover Color of the tab button font when the mouse over the tab.
--cd-background Color of the background tab button and bar.
--cd-background-selected Color of the background tab button when the tab is selected.
--cd-background-hover Color of the background tab button when the mouse over the tab.
CdTabButtonComponent Selector: cd-tab-button The CdTabButtonComponent component allow you to define the button of tabs for your implementation.

Properties

iconPosition
Description

String: define the icon position in button:

  • center: icon placed at center of button (default).
  • start: icon placed at first in button.
  • end: icon placed at end of button.

You shall define the icon first in your cd-tab-button.
ripple
Description

String: define the ripple effect when button is clicked:

  • default: no ripple effect applied.
  • unbounded: the unbounded ripple effect is applied.
  • bounded: the bounded ripple effect is applied.
routerLink
Description Array<String>: Define the link to activate by Angular Router.
selected
Description Boolean: activate (true) the tab at start-up if the parameter selectMode is set to config.
tab
Description String: define the tab id to activate when clicked.
CdTabContentComponent Selector: cd-tab-content The CdTabContentComponent component allow you to define the content of tabs for your implementation.

Properties

id
Description String: define the tab id linked to tab property of CdTabButtonComponent.
CdTabInterface Interface The CdTabInterface is the interface used for interaction with CdTabsComponent.

Fields

num Number: the tab index in list.
tabId String: the tab id.
tabButton CdTabButtonComponent: the tab button.
tabContent CdTabContentComponent: the tab content.
