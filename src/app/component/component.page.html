Components API

The CdTabsComponent component is organized under several sub-components:

  • CdTabBarComponent: used to organize the tab buttons.
  • CdTabButtonComponent: used to define the tab button content.
  • CdTabContentComponent: used to define the tab content.
CdTabsComponent Selector: cd-tabs The CdTabsComponent component is the entry point of the tabs definition.

Properties

selectMode
Description Set-up the autoselection mode at initialization:
  • config: use the configuration to auto select the activated tab (default).
  • router: use router path matching to auto select the activated tab.
displayMode
Description Set-up the display mode of tabs:
  • default: display tabs in default format.
  • segments: display tabs as Ionic Segments.
disposition
Description Set-up the disposition of tabs:
  • left: align tab buttons on left side (default).
  • center: align tab buttons on center side.
  • right: align tab buttons on right side.
  • vertical: display tab buttons in vertical bar.
