Angular Cd-Tabs !

Introduction

cd-tabs (CdTabsComponent) is a simple Angular CLI component that manage tabs, based on Ionic tabs.

There are the main features of this component:

  • horizontal and vertical tabs can be managed.
  • icons are managed.
  • Angular router or simple tab are managed.
  • not slotted component.

CdTabsComponent is a good way if you want to display a vertical menu without slot constraints.

About Ionic 4 and this component:

This component is designed due to limitation of ion-tabs, which use a web component slot and manage only bottom and top disposition.

This component not use slot and disposition is managed by a property.
