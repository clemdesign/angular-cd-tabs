Basic Usage

Colors

The cd-tabs component can use colors of Ionic Themes or custom colors.
Ionic colors In a Ionic app, you can use the input property color of CdTabBarComponent  . Refer to Ionic Integration section for demonstration. Custom colors In standalone app, you can customize the colors of CdTabBarComponent  by defining variables in a CSS class . Refer to Standalone Integration section for demonstration.

Vertical Tabs

With the input property disposition for CdTabsComponent , you can set a vertical tabs with the value vertical.

Rendering Code HTML Tab 1 Tab 2 Tab 1 is displayed ! Tab 2 is displayed !

Router integration

With the input property routerLink for CdTabButtonComponent , you can link the tab action to Angular Router.

You shall set the property selectMode for CdTabsComponent  to value router for using Router integration.
